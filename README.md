## Description
This project is a simple practice on using Postgres, it performs basic Create, Read, Update and Delete equests.

## Tech Stack
Node.js - Runtime environment <br>
Express - Framework <br>
PostgreSQL - Databsae
dotenv - hides sensitive info

## How tp run locally
Clone this repo and install all depedencies listed in the package.json file using npm install <br>
Fill in your environment variables as shown in the .env.example file
start the server by using node index.js on your command line or <br>
Install nodemon as a dev dependency, then use nodemon index.js to keep server automatically running <br>
You can now test all endpoints using Postman or Thunderclient.
